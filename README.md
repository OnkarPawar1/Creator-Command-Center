# 🚀 Creator Command Center

**Creator Command Center** is a high-performance, automated workflow pipeline designed for YouTube creators who specialize in book summary and educational content. It streamlines the complex 10-step process of turning a physical or digital book into a fully produced, high-quality YouTube video using the power of AI tools.

![Creator Command Center Preview](https://via.placeholder.com/1200x600/6366f1/ffffff?text=Creator+Command+Center+Dashboard)

## ✨ Features

- **Project Dashboard**: Manage multiple book video projects simultaneously.
- **Interactive 10-Step Workflow**: A guided path from book selection to YouTube publication.
- **AI Tool Integration**: Direct access to essential tools like NotebookLM, PDFSplicer, SynthesisLab, Claude, and Canva.
- **Progress Tracking**: Real-time progress bars and milestone celebrations for every project.
- **Smart Metadata Storage**: Dedicated areas for title, description, and tag generation.
- **Local Persistence**: All your data is saved automatically to your browser's local storage.

## 🛠️ The 10-Step Pipeline

1.  **Decide & Log the Book**: Start with a solid plan.
2.  **Download Book PDF**: Gather your source material.
3.  **Split PDF into Smaller Parts**: Optimized for AI processing via [PDFSplicer](https://onkarpawar1.github.io/PDFSplicer/).
4.  **Generate Audio (NotebookLM)**: Deep-dive AI audio overviews.
5.  **Download MP3 Audio Files**: Organize your narrative.
6.  **Generate Background Image**: Create visual ambiance with [SynthesisLab](https://onkarpawar1.github.io/SynthesisLab).
7.  **Convert Image to Video**: Add subtle motion using Gemini or Whisk.
8.  **Merge Audio & Video in Canva**: Professional editing made simple.
9.  **Generate YouTube Metadata**: AI-powered SEO optimization.
10. **Upload & Publish!**: Share your creation with the world. 🎉

## 📦 Installation & Setup

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/OnkarPawar1/Creator-Command-Center.git
    cd Creator-Command-Center
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🌐 Deployment

This project is configured for easy deployment to **GitHub Pages**.

```bash
npm run deploy
```
